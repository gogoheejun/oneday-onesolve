해설코드:
```java
public int[] solution(String s, char t) {
        int[] answer = new int[s.length()];
        int p = 100000;
        for(int i=0; i<s.length(); i++){
            if(s.charAt(i) == t){
                p = 0;
                answer[i] = p;
            }else{
                p++;
                answer[i] = p;
            }
        }

				p = 100000;
        for(int i=s.length()-1; i>=0; i--){
            if(s.charAt(i) == t){
                p = 0; 
								//어차피 그 자리에는 0이 있으니까 answer[i] = p 하지 않는다.
            }else{
                p++;
                answer[i] = Math.min(answer[i],p);
            }
        }
        return answer;
    }


```
